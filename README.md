# Expense Tracker (Flask)

A beginner-friendly expense tracker web application built using Python Flask and SQLite.

## Features
- Add daily expenses
- Categorize expenses
- View all expenses in a table

## Tech Stack
- Python
- Flask
- SQLite
- HTML, CSS

## How to Run
1. Install Python
2. Install Flask:
   python -m pip install flask
3. Run the app:
   python app.py
4. Open browser:
   http://127.0.0.1:5000
